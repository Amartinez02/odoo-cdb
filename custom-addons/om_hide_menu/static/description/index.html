Hide Menu User Wise

Hide specific menu items for individual users without modifying their group-based access rights. Control menu visibility on a per-user basis while preserving underlying permissions.

✨ What it does

  • ✅Provides user-wise menu visibility control through a simple configuration interface.
  • ✅Hide any menu item for specific users without affecting their access rights.
  • ✅Works independently of group permissions — users retain full access to underlying features.
  • ✅Changes take effect immediately after saving and refreshing the browser.

🛠 How it works

  • Adds a Hide Menus tab to the user form where you can select menus to hide.
  • Selected menus are filtered out from the user's visible menu tree.
  • Parent menus without visible children are automatically hidden.
  • Server-side enforcement ensures consistent behavior across all sessions.

⚙️ Configuration: User-wise Menu Control

  1. Install the module from Apps.
  2. Go to Settings → Users & Companies → Users.
  3. Open the user for whom you want to hide menus.
  4. Navigate to the Hide Menus tab.
  5. Add the menu items you want to hide for this user.
  6. Save and refresh the browser — changes take effect immediately.

Quick Tip: You can hide parent menus to automatically hide all their child menus, or hide individual sub-menus for more granular control.
